# Biometric-Doorlock-System

An Arduino based biometric doorlock system. 

# Prerequisites

* Arduino Uno 
* Solenoid Doorlock
* Transistor
* Adafruit Fingerprint Sensor
* 9v Batteries
* 5v/9v Adapter with Barrel Jack

# Installation

Upload the code to the Arduino board via computer and USB.
